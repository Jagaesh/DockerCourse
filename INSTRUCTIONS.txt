# Node.js Project Setup Instructions

**Initialize a Node.js Project**

1. Navigate to your app folder:
    ```bash
    cd ./DockerCourse/app
    ```

2. If you donâ€™t have a `package.json` file, create one by running:
    ```bash
    npm init -y
    ```

   Or if you already have a `package.json` file but want to update all dependencies:
    ```bash
    npm update
    ```

3. Install the Express & MongoDB frameworks using npm:
    ```bash
    npm install express
    npm install mongodb
    ```

4. Run the Server:
    ```bash
    node server.js
    ```
